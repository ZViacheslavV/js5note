## [JS5Note](https://zviacheslavv.github.io/js5note/)
> A tiny, educational project for creating and managing simple notes. 

---

## Tech Stack
- HTML, CSS, JS – core web technologies
- Vite – fast development server and build tool
- Libraries: nanoid (unique IDs), izitoast (notifications), modern-normalize
  
## Key Features
- Add, edit, and delete notes
- Unique ID generation for notes
- Theme changing
- Notifications for user actions
- Fast reload during development with Vite
